# TripLens UI And Forms

Use this skill when improving visual polish, dashboard layout, selected-trip
controls, comparison controls, New trip forms, or Edit trip forms.

## Dashboard Goals

- Make the selected trip obvious.
- Make the comparison trip a separate explicit choice.
- Keep overview metrics easy to scan.
- Keep charts paired with readable summaries.
- Use desktop space for grouping, not decoration.
- Keep mobile layout comfortable and linear.

## Form Goals

- Labels should be visible and close to the fields they describe.
- Category amount fields should be easy to scan and edit.
- Notes should not crowd the money inputs.
- Primary and secondary actions should be visually distinct.
- The form should not rely on tiny columns on mobile.
- Modal or drawer content should fit without broken spacing.

## Visual Direction

TripLens should feel light, spacious, and travel-oriented. Favor clean surfaces,
soft shadows, restrained blue/coral accents, clear section rhythm, and roughly
8px radii. Avoid turning the interface into a generic enterprise dashboard.

## Accessibility

- Buttons and selects need clear accessible names.
- Charts need text summaries.
- Focus and hover states should stay visible.
- Do not communicate status by color alone.
